# Payroll Management System

This is a simple payroll management system built with React and TypeScript. It allows users to add and manage employees, calculate gross and net salaries based on different payroll types, and store data using local storage.

## Features

- Add new employees with details such as name, position, and payroll type.
- View a list of employees in a table format.
- Edit and delete employee records.
- Calculate gross and net salaries based on the selected payroll type.
- Data is stored in local storage for persistence.

## Technologies Used

- React
- TypeScript
- Local Storage
- CSS for styling

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm (Node Package Manager)

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd payroll-management-system
   ```

3. Install the dependencies:
   ```
   npm install
   ```

### Running the Application

To start the application, run the following command:
```
npm start
```
This will start the development server and open the application in your default web browser.

### Usage

- Use the Employee Form to add new employees.
- The Employee List will display all added employees.
- You can edit or delete employees from the list.
- The Payroll Calculator will help you compute gross and net salaries.
